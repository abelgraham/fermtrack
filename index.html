Track dough, fermentation, and prep workflow — from mix to oven.

FermTrack is a multi-tenant kitchen system for tracking dough batches, fermentation stages, and corrective actions. Built for bakeries that want consistency without paper logs.

Open Live Demo See Features   Source Code
  Active development. Multi-tenant auth, admin panel, and ESP32 sensor integration available. Self-hostable.

What FermTrack does

Built around a pretzel and laugenbrezel workflow, but designed for any fermentation-heavy kitchen that needs clarity, consistency, and traceability.

01

Track every batch

Log batch ID, weight, time, and status so production never disappears into guesswork.

02

Multi-tenant bakery management

Each bakery gets its own isolated workspace with role-based access for bakers, managers, and admins.

03

Standardize workflow

Document fortifying, re-balling, degassing, washing, and other key actions in one system.

04

ESP32 sensor integration

Connect CO2, temperature, and humidity sensors for real-time fermentation monitoring with hardware you own.

Built with

A straightforward stack designed to be self-hostable on a single machine or deployed to the cloud.

Flask Backend

Python REST API with JWT auth and SQLite

Vanilla Frontend

No framework — plain HTML, CSS, and JavaScript

ESP32 Sensors

PlatformIO-based firmware for environmental monitoring

How it fits the kitchen

FermTrack is not just a recipe companion. It supports a more systematic kitchen by making fermentation and prep visible across the whole process.

  •   Create and label batches clearly
  •   Track fermentation stages across the day
  •   Record what needs to be fortified, re-balled, degassed, or washed
  •   Keep production decisions visible instead of locked in someone's head

"Built for cooks and bakers who want consistency without turning the kitchen into paperwork."

Long term, FermTrack can grow beyond batch logging into environmental tracking, automated alerts, and hardware-assisted fermentation monitoring — all self-hosted and open source.
